# Sleep Health & Lifestyle Analysis using Power BI

## Project Overview
This project analyzes the relationship between sleep health and lifestyle factors using an interactive Power BI dashboard. The analysis explores how factors such as stress levels, BMI, occupation, physical activity, and gender influence sleep duration, sleep quality, and sleep disorders.

The goal of this project is to transform raw health data into meaningful visual insights that help understand sleep patterns and lifestyle impacts.

---

## Problem Statement
Modern lifestyles involving high stress levels, irregular work schedules, and reduced physical activity are negatively affecting sleep quality. Many individuals suffer from sleep disorders such as insomnia and sleep apnea without understanding how lifestyle factors influence their sleep patterns.

This project uses data visualization to analyze sleep health data and identify important patterns and relationships.

---

## Objectives
- Analyze the relationship between lifestyle factors and sleep health.
- Identify patterns in sleep duration, stress levels, and physical activity.
- Study the impact of BMI, gender, and occupation on sleep quality.
- Visualize sleep disorder distribution across different groups.
- Develop an interactive Power BI dashboard for data exploration.
